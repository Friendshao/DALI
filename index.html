Dali

Non-Gaussian Likelihood Approximations

View on GitHub Download .zip Download .tar.gz

DALI is a fast approximation of non-Gaussian likelihoods. It extends the well-known Fisher Matrix in a straightforward way. The code is written in C/C++.

Documentation

The code has a Doxygen documentation and a handwritten manual. If the code gives you a hard time, please criticize the manuals before you criticize the code: the code had many, many test runs whereas the documentation had none. It is still subject to improvement and helpful criticism is very welcome.
